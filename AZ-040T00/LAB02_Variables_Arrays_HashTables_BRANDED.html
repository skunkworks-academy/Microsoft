AZ-040 Lab: Using Variables, Arrays & Hash Tables

One-Pager Study & Troubleshooting Guide

Key Objectives

  • Understand variable declaration and scope.
  • Create and manipulate arrays and hash tables.
  • Use common operators and methods for data management.

Quick Reference Commands

  • $var = "Value" — Define a variable.
  • $array = 1,2,3,4 — Create a simple array.
  • $array[0] — Access array element.
  • $hashtable = @{ Key="Value"; Name="John" } — Create a hash table.
  • $hashtable["Key"] — Retrieve value from hash table.
  • $array += 5 — Add item to array.

Troubleshooting & Root Cause Analysis

  • Variable not recognized: Ensure variable is declared in the correct scope (local vs global).
  • Array indexing error: Verify index is within bounds of array length.
  • Hashtable key issues: Check spelling and case sensitivity for keys.

Best Practices

  • Use descriptive names for variables to improve readability.
  • Initialize arrays and hash tables explicitly for predictable results.
  • Leverage Get-Variable and Remove-Variable for management.
  • Validate inputs before adding to arrays or hash tables to prevent runtime errors.

‚ö° Quick Reference Commands

# Local users and groups
Get-LocalUser
New-LocalUser -Name "TestUser" -Password (Read-Host -AsSecureString)
Add-LocalGroupMember -Group "Administrators" -Member "TestUser"

# Services
Get-Service
Start-Service -Name "Spooler"
Stop-Service -Name "Spooler"
Set-Service -Name "Spooler" -StartupType Automatic

# Processes
Get-Process
Stop-Process -Name notepad -Force

# Event Logs
Get-EventLog -LogName System -Newest 20
Clear-EventLog -LogName Application

üõ†Ô∏è Troubleshooting & RCA

  • Permission denied: Run PowerShell as Administrator for system-level commands.
  • User not found: Verify the spelling and existence of the account with Get-LocalUser.
  • Service control issues: Check dependencies with Get-Service -DependentServices.
  • Locked processes: Use Stop-Process -Id <PID> -Force if the name fails.
  • Event log clearance blocked: Ensure audit/retention policies allow log clearance.

‚úÖ Verification Steps

  • Run Get-LocalUser to confirm user creation or changes.
  • Use Get-Service -Name Spooler to confirm service status updates.
  • Run Get-Process to validate process management actions.
  • Query logs with Get-EventLog to confirm log entries were updated/cleared.
